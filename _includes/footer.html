About

We are a group of people who have passion in the open source code community. Our goal for this service is to provide the greatest possible convenience to developers so that they don’t have to set up a personal web server just to store and to showcase their projects online.

WordPress Plugin

Are you running a WordPress site? We got You! Statically service can be used on WordPress site too. With this plugin, you’ll be able to:

  • Replace WordPress library with Statically CDN.
  • Load images and compress it with Statically Imgpx.
  • Edit and crop images on the fly.
  • Reduce origin server loads.
